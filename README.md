# 📊 Aplicación de análisis exploratorio de datos con Streamlit

---

## 📝 Descripción del proyecto

Este proyecto consiste en el desarrollo de una **aplicación web interactiva para realizar Análisis Exploratorio de Datos (EDA)** utilizando **Python y Streamlit**.

La aplicación permite explorar un dataset de forma visual y sencilla, proporcionando herramientas para entender mejor la estructura de los datos antes de aplicar modelos de machine learning o realizar análisis más avanzados.

Entre las funcionalidades principales se incluyen:

- Visualización de las primeras filas del dataset
- Análisis de estadísticas descriptivas
- Identificación y visualización de valores faltantes
- Gráficos para analizar distribuciones de variables
- Exploración de variables numéricas y categóricas

La aplicación facilita realizar un **análisis exploratorio rápido e interactivo directamente desde el navegador**.

3. Ejecutar la aplicación:
   streamlit run app.py
